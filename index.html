Welcome to the Japanese OPNsense Translation project.

This fork was created only to begin the work necessary to translate OPNsense into the Japanese language, we work by following OPNsense core, and submitting Pull Requests to them in GitHub.

Contributions, BUGS, and Help

Discussions regarding this work should happen in the OPNsense Japanese forum! https://forum.opnsense.org/

Contributor Quick-start

A quick-start to get editing the JP translation,

$ git clone git@github.com:dotike/opnsense.core.ja_JP.UTF8.git
$ cd opnsense.core.ja_JP.UTF8.git
$ git checkout -b master origin/master
$ vim src/share/locale/ja_JP/LC_MESSAGES/OPNsense.po

Everyone can help,

If you are Japanese/English bilingual, you can just start working on your own by checking out the code! One warning, keep your commits and work short- it would be very sad if someone else was translating the very same lines you were- and time was wasted!

$ git https://github.com/dotike/opnsense.core.ja_JP.UTF8.git
$ cd opnsense.core.ja_JP.UTF8.git
$ git checkout -b master origin/master
$ vim src/share/locale/ja_JP/LC_MESSAGES/OPNsense.po
(translate some lines)
$ git format-patch <commithash> src/share/locale/ja_JP/LC_MESSAGES/OPNsense.po
(email the resulting patch files to ike at blackskyresearch dot net!)

Once you have submitted a patch or two, feel free to ask for repo access to make things easier- (or just send us git formatted patch files to get going!)
